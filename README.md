# n8n 超級個體包 — 新手實戰學習站

> 專為創業者、一人公司與中小企業主打造的 n8n 自動化模板學習平台

🔗 **[立即訪問學習站](https://darkschen.github.io/n8n-super-individual-learning/)**

## 關於

這是 n8n 超級個體包的免費學習網站，包含：

- 🛠️ **環境部署教學** — 從零開始架設你的 n8n 伺服器
- 🚀 **五大核心模板設定** — 跟著影片一步步完成
- 💎 **進階資源** — 自學筆記、API 教學、免費額度

所有教學內容免費公開，模板為付費商品。

## 技術

- [Vite](https://vitejs.dev/) + TypeScript
- 純靜態網站，零後端
- GitHub Pages 自動部署
